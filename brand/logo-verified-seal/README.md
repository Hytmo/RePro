# RePro — Verified Seal · Logo Kit

Production logo package for concept 05 (the Verified Seal).

**Start here:** `repro-seal-brand-sheet.png` — the full kit on one page.

- repro-seal-horizontal-tagline.svg . horizontal + "réputation professionnelle"
- repro-seal-on-dark.svg ............ light wordmark for dark backgrounds
- repro-seal-horizontal-on-dark.svg . horizontal, light wordmark
- repro-seal-mark.svg ............... seal only (no wordmark)
- repro-seal-wordmark.svg ........... wordmark only
- repro-seal-mono-blue/black/white.svg  one-colour knockout versions
- repro-seal-favicon.svg ............ simplified app/browser icon

## Colours
RePro Blue #1D4ED8 · Deep Blue #1E3A8A · Sky #93C5FD · Ink #0B1B33 · Paper #F4F1E9
